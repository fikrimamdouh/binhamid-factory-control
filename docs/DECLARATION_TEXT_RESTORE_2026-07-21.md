# استعادة نصوص الإقرارات الأصلية

- المصدر المرجعي: نصوص الإقرارات السابقة قبل استبدال صياغة محفظة العملاء.
- أُعيدت نصوص: المركبات، محفظة العملاء، مسؤولية الائتمان، العهدة النقدية، محطة الخلط، المسؤولية الفنية، الورشة، والإقرار الختامي.
- أضيف بند واحد فقط إلى محفظة العملاء يجمع متابعة المتأخرات وشرط توفر السيولة/الدفعة المقدمة.
- لا تُعدّل هذه المعالجة العملاء أو الأرصدة أو المبيعات أو التحصيلات أو قاعدة البيانات.
- يستخدم الموقع وتيليجرام مصدرًا قانونيًا متطابقًا لنص محفظة العملاء.
- رقم إصدار النص: `2026-07-21-original-plus-liquidity-v1`.
